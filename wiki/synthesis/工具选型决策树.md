# 工具选型决策树

本页是综合分析页，用于把标准页和 15 个案例页中的共性路径整理成现场选型线索。它不是独立标准，最终工具规格、扣型、许用拉力、许用扭矩和管柱组合必须回到 [[打捞工具目录]]、[[解卡操作规程]]、[[水平井特殊工艺]] 以及现场设计复核。

## 输入信息

处置前至少确认以下信息：

| 字段 | 用途 | 缺失影响 |
| --- | --- | --- |
| 井型与井眼轨迹 | 判断直井、定向井、水平井差异 | 无法判断摩阻、携砂和工具通过性 |
| 鱼顶深度 | 计算管柱长度、作业窗口和风险 | 无法校核工具下入与循环路径 |
| 落鱼类型与尺寸 | 确定内捞、外捞、造扣或套铣路线 | 工具规格只能写为待确认 |
| 扣型与内外径 | 选择捞矛、公锥、母锥、打捞筒及转换接头 | 不得直接给出确定型号 |
| 是否砂埋或强阻卡 | 决定先清洁井眼还是直接打捞 | 直接打捞可能扩大事故 |
| 已采取措施 | 判断是否进入升级工艺 | 避免重复无效作业 |

## 总体路径

```text
确认井控与循环条件
→ 复核鱼顶、落鱼、扣型、井斜和钻井液
→ 判断是否砂埋/遇阻/强卡
→ 先清洁或解卡
→ 再选择内捞、外捞、造扣打捞
→ 失败后升级震击、套铣、磨铣
→ 多轮失败或风险升高时论证侧钻/弃鱼
```

## 决策树

### 1. 井况是否稳定

| 判断 | 推荐动作 | 依据 |
| --- | --- | --- |
| 井控、循环、悬重、扭矩正常 | 进入鱼顶复核与工具选型 | [[解卡操作规程]] |
| 循环不畅、返砂异常、疑似砂埋 | 先循环洗井或冲砂，不直接造扣 | [[解卡操作规程]]、[[水平井特殊工艺]] |
| 井控异常或工具遇阻加重 | 暂停打捞，先处理井控和井筒安全 | [[风险评估矩阵]] |

### 2. 落鱼类型为钻杆或油管

| 鱼顶状态 | 优先工具/工艺 | 说明 |
| --- | --- | --- |
| 内孔通畅、扣型可确认 | 打捞矛、可退式内捞工具 | 需按内径、扣型、许用载荷复核，见 [[打捞工具目录]] |
| 外部可抓、鱼顶规则 | 打捞筒、卡瓦类外捞工具 | 需确认外径、鱼顶形态和井眼间隙 |
| 扣型未知但可造扣 | 公锥或母锥类造扣工具 | 造扣参数不得凭空给出，需现场设计确认 |
| 疑似砂埋或鱼顶被覆盖 | 先洗井/冲砂，必要时套铣后再打捞 | 见 [[解卡操作规程]]、[[水平井特殊工艺]] |

### 3. 落鱼类型为钻铤、接头或大直径工具

| 情况 | 推荐路线 | 风险控制 |
| --- | --- | --- |
| 鱼顶规则、外径可抓 | 外捞工具优先 | 校核井眼间隙和工具通过性 |
| 重量大、阻卡明显 | 先活动、震击或解卡，再打捞 | 上提负荷不得超过强度校核值 |
| 鱼顶损坏或无法抓取 | 套铣、磨铣、修鱼顶后再打捞 | 监测扭矩、返砂、循环和金属屑 |

### 4. 水平井或大斜度井

水平井不宜简单套用直井路径。选型时增加以下判断：

- 工具通过性：检查斜井段、水平段、狗腿度、扶正器和工具外径。
- 携砂能力：先评估循环排量、返砂和沉砂风险。
- 摩阻风险：活动管柱、震击和上提载荷应结合轨迹和管柱强度确认。
- 可退能力：优先考虑可退式工具，避免二次卡钻。

相关页面：[[水平井特殊工艺]]。

## 与案例的连接

案例匹配时优先比较：

1. 落鱼类型是否同为钻杆、钻铤或接头。
2. 鱼顶深度是否接近。
3. 是否出现上提无效、砂埋、循环异常、套铣、磨铣、震击等关键词。
4. 最终结果是成功打捞、失败、侧钻还是弃鱼。

## 输出要求

Agent 生成方案时，工具选型必须写成：

```text
推荐工具：工具类别 + 待现场确认的规格
确认依据：落鱼内外径、扣型、鱼顶形态、井眼轨迹、标准适用范围
禁止写法：直接给出无来源的精确型号、扭矩、拉力或震击参数
```
