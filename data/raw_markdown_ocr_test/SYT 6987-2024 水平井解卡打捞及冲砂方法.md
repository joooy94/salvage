4施工设计要求
4.1
施工设计编写应以地质设计和工程（工艺）设计为依据。
4.2
应查阅并史，了解历次作业情况及目前并下技术状况，收集井身结构、并眼轨迹数据、落鱼状
况和砂柱高度。
4. 3
应现场勘查井场及其周围环境。
4. 4
施工设计应根据斜井段和水平段的下井管柱结构、规格型号、起下速度提出施工要求。
4. 5
施工设计中应注明在最大井眼曲率条件下可通过管柱及工具的安全尺寸。
4.6
施工设计的编写依据、内容要求、审批程序与设计书格式应符合SY/T6264的规定。
